Kratos example using golang
==========================================

Run the example with `docker compose up --build`. Once it has finished building head over to `http://localhost:8081` on your browser to register an account.

Most of the logic is in assets/kratos.html where we convert the data from Kratos into html.

This example was written for my [blog post](https://www.hekewa.com/blog/ory-kratos-go-backend).
